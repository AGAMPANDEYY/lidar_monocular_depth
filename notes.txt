6. Latest to run main.py and evaluate the results 


To get more extensive results, you should:

Increase the number of frames processed:

python main.py --ecw_use_fused --max_frames 100

Run with different depth backends to compare:

python main.py --ecw_use_fused --depth_backend zoe --max_frames 100
python main.py --ecw_use_fused --depth_backend midas --max_frames 100
python main.py --ecw_use_fused --depth_backend fastdepth --max_frames 100

Then evaluate all results:

python eval/compare_fusion.py --data_dir data/fused_output
python eval/save_and_visualize.py --data_dir data/fused_output
python Eval/ablations.py --data_dir data/fused_output

This will give you comprehensive tables showing:

Depth accuracy improvements with fusion
ECW detection performance
Pipeline timing across different backends
Object-level depth statistics
